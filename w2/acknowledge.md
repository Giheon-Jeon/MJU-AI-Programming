# 2주차 학습 내용

## 주제
파이썬 기초 I-1, I-2 (변수, 자료형, 연산자, 문자열, 리스트, 조건문)

## 개념 요약
- **변수와 리터럴**: `print()`, `int`, `float`, `str`, `bool`. 리터럴은 코드에 쓴 데이터 그 자체(상수), 변수는 값을 이름으로 저장하는 방법 (할당 연산자 `=`).
- **f-string**: 문자열 안에 변수 값을 간편하게 삽입할 때 사용 (ex. `f"결과는 {value}입니다."`).
- **타입 변환(Casting)**: `int()`, `float()`, `str()` - 항상 연산 전 올바른 자료형인지 `type()`으로 확인.
- **주석(Comment)**: `#` 은 한 줄 주석, `"""` 세 개는 멀티라인 문자열 및 독스트링에 사용. (예: '내일의 나'를 위한 설명)
- **문자열(String)**: 
  - 기본: 작은 따옴표(`'`), 큰 따옴표(`"`), 삼중 따옴표(`"""`).
  - 더하기 `+` 로 연결 가능, `len()`으로 길이 확인.
  - 불변(immutable): 인덱싱(`word[0]`)으로 읽을 수는 있으나 직접 수정 불가.
  - 슬라이싱 `[start:end:step]`: 원하는 범위 자르기, `[::-1]`로 문자열 뒤집기.
  - 메서드: `replace()`, `split()`, `count()`, `strip()`, `upper()`, `isdigit()`, `index()`, `endswith()`.
- **리스트(List)**: 
  - `[]` 로 생성, 순서가 있고 가변적(mutable).
  - 메서드: `append()`, `insert()`, `remove()`, `pop()`, `count()`, `sort()`, `reverse()`.
  - 내장 집계 함수: `sum()`, `min()`, `max()`, `len()`.
- **튜플(Tuple), 세트(Set), 딕셔너리(Dictionary)**:
  - 튜플 `()`: 순서가 있지만 불변(immutable). 패킹/언패킹에 유리.
  - 세트 `{}`: 순서 없고 중복 불가. 교집합(`&`), 합집합(`|`) 연산 등 가능.
  - 딕셔너리 `{key: value}`: 키-값 쌍, 키 중복 불가, 순서 없음 (Python 3.7+부터는 삽입순서가 유지되나 원칙상 인덱싱은 키로).
- **입력받기**: `input()` 함수. 기본적으로 문자열(`str`)로 입력받으므로 숫자 연산 시 `int()`, `float()` 로 변환 필수. 다수 입력 시 `.split()` 연계.
- **제어 흐름 (if-elif-else)**: 참(`True`) / 거짓(`False`) 비교 연산결과에 따라 코드 실행을 분기. Indentation(들여쓰기)가 매우 중요.

## 기억해야 할 꿀팁
- 문자열이나 리스트의 인덱스는 **0부터 시작**하며, 음수 인덱스 `-1`은 맨 마지막 요소를 가리킴. 슬라이싱 시 `end` 범위는 **포함하지 않고(미만)** 자름.
- **List 집계**: 정렬 후 `scores[0]`(최솟값), `scores[-1]`(최댓값)을 써도 되지만, 기본 제공 `min()`, `max()` 함수 사용이 편리.
- 사용자 조건 분기처리 시 `print()` 등을 활용해 **중간값**과 **타입(`type()`)**을 꼭 점검하고, 에러나면 **자료형**을 가장 먼저 의심할 것!

## 코드 스니펫
```python
# 문자열 메서드 예시
text = "naver.com"
print(text.replace("naver", "google")) # "google.com"

# if 조건문 분기 + 리스트
scores = [88, 76, 93, 100, 69]
scores.sort()
lowest = scores[0]

price = 45000
if price < 30000:
    print("불가")
elif price < 50000:
    price += 2500
else:
    print("무료배송")
print(f"최종 금액: {price}")
```
