# 🤖 MJU-AI-Programming (2026-1)

2026학년도 1학기 명지대학교 AI 프로그래밍 수업 실습 및 학습 내용을 정리하는 저장소임. 기초 파이썬 문법부터 데이터 분석 라이브러리(NumPy, Pandas) 활용까지 다룸.

## 📂 저장소 구조
- `.agents/`: AI 튜터 시스템 설정 및 가이드라인
- `w1~w7/`: 주차별 학습 지식(`knowledge.md`) 및 실습 코드
- `과제/`: 수업 관련 과제 제출 및 연습 파일
- `status.md`: 전체 학습 진도 및 학생 성취도 기록부
- `main.py`: 현재 진행 중인 주요 로직 테스트 파일
- `*.csv`, `*.pkl`: 실습에 사용되는 데이터 파일들

## 🗓️ 주차별 학습 요약
- **1주차**: AI 프로그래밍 환경 세팅. Anaconda 가상환경 및 Antigravity 에이전트 설정 완료함.
- **2주차**: 파이썬 기초 문법 I (변수, 자료형, 연산자, 제어문).
- **3주차**: 파이썬 기초 문법 II (함수, 모듈, 클래스/객체지향 기초).
- **4주차**: 디버깅 기법 및 파일 입출력. `with` 구문, CSV/Pickle 파일 처리 학습함.
- **6주차**: 데이터 분석 기초. NumPy 배열 생성 및 기본 연산 이해함.
- **7주차**: NumPy 심화. Axis(축) 개념, 브로드캐스팅 규칙, 행렬 연산 마스터함.

## 🚀 주요 특징
- **AI Tutor 연동**: 전용 프롬프트(`tutor.md`)를 통해 1:1 맞춤형 피드백을 받으며 학습 중임.
- **지식 자산화**: 매주 배운 내용을 `knowledge.md`에 문서화하여 나만의 데이터 사이언스 위키를 구축함.
- **데이터 시각화**: Matplotlib 등을 활용하여 분석 결과를 시각화하는 실습 병행 중임.
