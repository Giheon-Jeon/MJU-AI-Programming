## 2. 주요 개념 요약

### 2.1. 변수와 명명 규칙 (Variables & Naming)
- **변수(Variable):** 데이터를 저장하고 재사용하기 위한 이름표입니다. `=` 연산자를 사용하여 값을 할당(Assign)합니다.
- **명명 규칙:**
  - 문자와 숫자, 언더바(`_`)를 사용할 수 있습니다. 숫자로 시작할 수 없습니다.
  - 공백을 포함할 수 없습니다 (예: `average score` [X]).
  - 파이썬 예약어(if, for, class 등)는 사용할 수 없습니다.
  - **Snake Case:** 단어 사이를 언더바(`_`)로 연결합니다. (예: `student_count`)
- **f-string:** 문자열 안에 변수 값을 간편하게 삽입할 때 사용 (예: `f"결과는 {value}입니다."`).

### 2.2. 자료형과 형변환 (Data Types & Type Conversion)
- `int`: 정수 / `float`: 실수 / `str`: 문자열 / `bool`: 불리언 (True, False).
- `type()`: 데이터의 타입을 확인하는 함수입니다.
- **형변환(Casting):** `int()`, `float()`, `str()` 등을 사용하여 명시적으로 타입을 변경할 수 있습니다. 연산 전 올바른 자료형인지 확인하는 습관이 중요합니다.

### 2.3. 문자열(String) 및 슬라이싱
- **기본:** 작은 따옴표(`'`), 큰 따옴표(`"`), 삼중 따옴표(`"""` - 멀티라인용) 사용.
- **불변성(Immutable):** 인덱싱(`word[0]`)으로 읽을 수는 있으나 직접 수정은 불가합니다.
- **슬라이싱:** `[start:end:step]` 형식을 사용하며, `end` 범위는 포함하지 않습니다.
  - `[::-1]`은 문자열을 뒤집는 유용한 표현입니다.
- **주요 메서드:** `replace()`, `split()`, `count()`, `strip()`, `upper()`, `isdigit()`, `index()`, `endswith()`.

### 2.4. 데이터 구조 (Data Structures)
- **List `[]`:** 순서가 있고 수정이 가능한(mutable) 목록.
  - 메서드: `append()`, `insert()`, `remove()`, `pop()`, `sort()`, `reverse()`.
- **Tuple `()`:** 순서가 있으나 수정이 불가능한(immutable) 목록. 패킹/언패킹에 유리.
- **Set `{}`:** 중복 불가, 순서 없음. 집합 연산(`&`, `|`) 가능.
- **Dict `{key: value}`:** 키-값 쌍 구조. 키는 중복될 수 없습니다.

### 2.5. 제어 흐름 및 함수
- **제어 흐름 (if-elif-else):** 조건의 참/거짓에 따라 실행을 분기합니다. 들여쓰기(Indentation)가 문법적으로 매우 중요합니다.
- **주요 내장 함수:** `abs()`, `round()`, `len()`, `sum()`, `min()`, `max()`.
- **모듈 활용:** `math.sqrt()` (제곱근), `random.randint(a, b)` (정수 난수) 등.

## 3. 코드 스니펫 및 뼈대 코드

### f-string 및 기본 변수
```python
name = "학생"
score = 95
print(f"{name}님의 점수는 {score}점입니다.")
```

### 문자열 메서드 및 슬라이싱 예시
```python
text = "naver.com"
print(text.replace("naver", "google")) # "google.com"
print(text[:text.index('.')])          # "naver"
```

### 리스트와 조건문 활용
```python
scores = [88, 76, 93, 100, 69]
scores.sort()
if scores[-1] >= 90:
    print("최고점은 90점 이상입니다.")
```

## 4. 기억해야 할 꿀팁 (Pro-Tips)
- **0부터 시작:** 문자열과 리스트의 인덱스는 항상 0부터 시작합니다.
- **-1 인덱스:** 맨 마지막 요소를 가리킬 때 매우 편리합니다.
- **디버깅 습관:** 에러가 발생하면 `type()`함수를 사용해 자료형을 가장 먼저 점검하고, `print()`로 중간값을 확인하세요.
